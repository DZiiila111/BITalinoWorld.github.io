Starts a signal acquisition from the device.

Parameters
[in]samplingRateSampling rate in Hz. Accepted values are 1, 10, 100 or 1000 Hz. Default value is 1000 Hz.
[in]channelsSet of channels to acquire. Accepted channels are 0, 1, 2, 3, 4, and 5. If this set is empty or if it is not given, all 6 analog channels will be acquired.
Note
This method cannot be called during an acquisition.

void BITalino::trigger ( const Vbool &  digitalOutput = Vbool())

Sets the digital outputs states.

Parameters
[in]digitalOutputVector of booleans to assign to digital outputs, starting at output 0. This vector cannot contain more than 4 elements. If the vector contains less than 4 elements, the remaining outputs are set to low state. If this parameter is not given, all digital outputs are set to low state.
Note
This method must be called only during an acquisition.
